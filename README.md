# 俄罗斯方块

一个基于HTML/CSS/JavaScript的经典俄罗斯方块游戏。

## 功能特性

- 经典的俄罗斯方块玩法
- 7种不同的方块形状(I, J, L, O, S, T, Z)
- 下一个方块预览
- 分数和等级系统
- 美观的渐变UI设计
- 暂停/继续功能

## 如何游玩

1. 点击"开始游戏"按钮开始
2. 使用键盘控制方块:
   - `<kbd>←</kbd>` `<kbd>→</kbd>` - 左右移动
   - `<kbd>↑</kbd>` - 旋转方块
   - `<kbd>↓</kbd>` - 加速下落
   - `<kbd>Space</kbd>` - 瞬间下落
   - `<kbd>P</kbd>` - 暂停/继续

## 游戏规则

- 消除完整的行可以获得分数
- 消除的行数越多，得分越高
- 分数越高，等级越高，下落速度越快
- 当方块堆叠到顶部时游戏结束

## 技术栈

- HTML5
- CSS3 (Flexbox, Grid, Gradient)
- JavaScript (ES6+)
